# WindShield

Frontend application for the Wallet Manager experience.

## Current scope

- Frontend-only app (no backend integration yet)
- Local state-driven wallet, cards, debt, and planning flows
- Prepared for backend wiring and UX iteration

## Local development

```bash
npm install
npm run dev
```

Backend (separate terminal):

```bash
npm run dev:backend
```

## Production build

```bash
npm run build
npm run build:backend
npm run preview
```
